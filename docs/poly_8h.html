Calculates value of polynomial in point x.

It's done by substituting first (main) variable of polynomial with x. The result may be a polynomial if coefficients of p are polynomials. If it is then te indexes of variables are decremented by 1. Formally for polynomial \(p(x_0, x_1, x_2, \ldots)\) the result is a polynomial \(p(x, x_0, x_1, \ldots)\).

◆ PolyDegBy()

poly_exp_t PolyDegBy ( const Poly *  p,
unsigned  var_idx 
)

Determines the degree of polynomial with respect to the given variable (-1 for zero polynomial).

Variables are indexed starting from 0. Variable with 0 index means the main variable of that polynomial. Higher numbers means the variables contained in coefficients of that polynomial.

You can understand it as recrusrion. All variables in polynomial coefficients that are not nested in any others are these with 0 index. All nested directly in them are these with numbered with 1. All nexted in directly them are these ... with 2. etc.

For variables index higher than degree of polynomial 0 is returned

  • or -1 if the polynomial itself is a zero one (polynomial is const for that variable as in normal math)
Parameters
[in]p: polynomial
[in]var_idx: index of variable
Returns
degree of polynomial p with respect to variable with index var_idx

◆ PolySprintf()

void PolySprintf ( char *  dest,
const Poly *  p 
)

Converts polynomial p to human-readable char sequence.

The result is saved in array pointed by dest.

Polynomials are converted using the following assumptions:

  • print only fully simplified terms of polynomial of form \(C*x_0^{a_0}*x_1^{a_1}*\ldots*x_n^{a_n}\)
  • encode variables using charactes a..z Variable with index 0 is a, 1 is b etc.
  • encode math operators in ascii manner e.g x^a * x^b
  • skip 1 wherever possible
  • do not print terms multiplied by 0
  • do not print sign wherever possible
Parameters
[in]dest: char array pointer
[in]p: polynomial
