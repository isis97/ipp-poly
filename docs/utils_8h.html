Detailed Description

Various utilities.

(C99 standard)

THIS FILE SHOULD BE ALWAYS INCLUDED AS FIRST ONE! This is only in case of compilation with UNIT_TESTING Because then all important definitions are replaced with a traps used for testing.

The unit tests for the library take advantage of UNIT_TESTING macro.

If you DO NOT WANT the traps to be emmited then define DISABLE_TRAPS macro before including this file :)

Unit tests should also define UNIT_TEST it marks the file no as part of libary to be tested but rater an external code. It works just as DISABLE_TRAPS macro. :)
