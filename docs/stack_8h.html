◆ StackAllocator

typedef StackData(* StackAllocator) (StackData data)

Function allocating/deallocating space for stack elements.

If the function is called as deallocator then it should free given data. Then should return NULL.

If as allocator then it should allocate new StackData. Assign a value of data to it. And return it.
