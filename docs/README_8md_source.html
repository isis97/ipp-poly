51 # Polynomial library
52 
53 All the documentation is provided in header files.
54 
55 There you've got example usage of the library:
56 
57 ```c
58  #include "poly.h"
59 
60  int main(void) {
61 
62  Poly p = PolyC(4);
63  // Create constant polynomial
64  // p(x) = 4
65 
66  Poly q = PolyP( PolyC(1), 2 );
67  // Create polynomial that consists of
68  // variable square multiplied by
69  // coefficient 1
70  // q(x) = 1*x^2
71 
72  // As you may see the coefficients that are placed
73  // next to variables may be other polynomials
74  // (polynomials from other variable)
75 
76  Poly r = PolyP( PolyP( PolyC(1), 2 ), 3 );
77  // In this example
78  // r(x,y) = (1*y^2)*(x^3)
79 
80  // Now you can do arithemtic operations on them
81  // for example addition:
82  Poly sum = PolyAdd(&p, &q);
83  // sum(x) = 4 + x^2
84  // If you are not sure what would happen if we sum
85  // q and r then we say that the result is
86  //
87  // q(x) + r(x,y) = 1*x^2 + (1*y^2)*(x^3)
88  // that is:
89  // x^2 + (y^2)(x^3)
90  //
91 
92  // Or evalue at specific point:
93  Poly rAtTwo = PolyAt(&r, 2);
94  // Now we assume the first variable is equal to 2
95  // So we calculate r(x,y) for x->2
96  // rAtTwo = r(2,y) = (2^3)*y^2 = 8*y^2
97 
98  // You can print them nicely:
99  PolyPrint(&rAtTwo);
100  // ^ Do it yourself and see what the output is
101 
102  // After everything we must free them all
103  PolyDestroy(&p);
104  PolyDestroy(&q);
105  PolyDestroy(&r);
106  PolyDestroy(&sum);
107  PolyDestroy(&rAtTwo);
108 
109  }
110 ```

112 # Polynomial calculator
113 
114 The calulator resides in `src/calc_poly.c`.
115 
116  It reads data from stdin line by line and:
117 
118 1. If the line starts with lower/upper case letter then it's assumed<br>to be an input command.
119 
120 2. Otherwise the input is parsed as a polynomial<br>The format of the polynomial is<br>`(COEFF_1,EXP_1)+(COEFF_2,EXP_2)+...+(COEFF_N,EXP_N)`<br>Where pair `(COEFF_N,EXP_N)` represents single monomial `(COEFF_N)*x^(EXP_N)`<br>The coefficients can be numbers:<br>`(1,3)+(2,4)` -> `x^3 + 2x^4`<br>Or polynomials (in this case they depend on other variable):<br>`((1,3)+(2,4),2)+(1,3)` -> `(x^3 + 2x^4)*y^2 + y^3`<br>can also contain monomials with the same exponent:<br>`(1,2)+(1,2)` -> 2x^2<br>Or can be just numbers:<br>`42` -> constant polynomial 42
121 
122 
123 All the polynomials are parsed and placed on top of the stack.
124 Then you can call one or more of the given operations
125 
126 to modify the stack contents:
127 
128 | Command | Parameters | Required stack size | Description |
129 |------------|------------|---------------------|-------------|
130 |**ZERO** | | 0 | Pushes zero const polynomial onto stack |
131 |**IS_COEFF**| | 1 | Checks if the stack top is a constant polynomial? |
132 |**IS_ZERO** | | 1 | Checks if the stack top is a polynomial equal to 0? |
133 |**CLONE** | | 1 | Copies the polynomial on top of the stack and places it on the top. |
134 |**ADD** | | 2 | Adds two polynomials from the top of the stack.<br>Then puts the result on the stack top. |
135 |**MUL** | | 2 | Multiplies two polynomials from the top of the stack.<br>Then puts the result on the stack top. |
136 |**NEG** | | 1 | Changes the sign of the polynomial on the top of the stack. |
137 |**SUB** | | 2 | Substracts two polynomials from the top of the stack.<br>Then puts the result on the stack top. |
138 |**IS_EQ** | | 2 | Checks if two top-most polynomials are equal. |
139 |**DEG** | | 1 | Checks the degree of the top-most polynomial.<br>Degree of the polynomial is a highest exponent of variable<br>encountered in the polynomial (assuming all variables are the<br>same one).<br>E.g. degree(x^2 + yx^2) = 3 |
140 |**DEG_BY** | *index* | 1 | Checks the degree of the top-most polynomial<br>with respect to the given variable. Variables are indexed from 0<br>(0 means the main variable of polynomial).<br><br>E.g.<br>`DegBy( ((1,2),3), 0 ) = DegBy( x^2 * y^3, 0 ) = 3`<br>`DegBy( ((1,2),3), 1 ) = DegBy( x^2 * y^3, 1 ) = 2` |
141 |**AT** | *value* | 1 | Evaluates the top-most polynomial in specified point<br>and put it into stack.<br>The polynomial is evalued for `MAIN_VARIABLE=[value]` where<br>`MAIN_VARIABLE` is variable with index 0.<br><br>E.g.<br>`At( ((1,2),3), 0 ) = At( x^2 * y^3, 0 ) = 0`<br>`At( (1,2), 2 ) = At( x^2, 2 ) = 4`<br><br>The result of this command is always a polynomial of degree one smaller<br>than the polynomial given. |
142 |**PRINT** | | 1 | Prints the top-most polynomial. |
143 |**POP** | | 1 | Pops the top-most polynomial from the stack. |
144 |**POW** | *exp* | 1 | Calculates the top-most polynomial power and push into the<br>stack. Obviously *exp* can be only a number! |
145 |**COMPOSE** | *count* | *count*+1 | Takes top-most polynomail from the stack.<br>(We will call it P)<br>Then take *count* polynomials from the stack (let's call them Q1, Q2 ...).<br>Then we know that `P = C_1*x_1^E_1 + C_2*x_2^E_2 + ...`<br>so we substitute<br>`x_1 -> Q1`<br>`x_2 -> Q2`<br>etc.<br>if the `x_n` has got no matching `QN` then we assume `x_n -> 0`<br><br>Then we put result of such substitution onto the stack. |
146 |**DUMP** | | 0 | Prints the stack contents. |
147 |**CLEAN** | | 0 | Clears the stack entinerely. |
148 |**EXIT** | | 0 | Force exits the calculator. |
