51 # Polynomial library
52 
53 All the documentation is provided in header files.
54 
55 There you've got example usage of the library:
56 
57 ```c
58  #include "poly.h"
59 
60  int main(void) {
61 
62  Poly p = PolyC(4);
63  // Create constant polynomial
64  // p(x) = 4
65 
66  Poly q = PolyP( PolyC(1), 2 );
67  // Create polynomial that consists of
68  // variable square multiplied by
69  // coefficient 1
70  // q(x) = 1*x^2
71 
72  // As you may see the coefficients that are placed
73  // next to variables may be other polynomials
74  // (polynomials from other variable)
75 
76  Poly r = PolyP( PolyP( PolyC(1), 2 ), 3 );
77  // In this example
78  // r(x,y) = (1*y^2)*(x^3)
79 
80  // Now you can do arithemtic operations on them
81  // for example addition:
82  Poly sum = PolyAdd(&p, &q);
83  // sum(x) = 4 + x^2
84  // If you are not sure what would happen if we sum
85  // q and r then we say that the result is
86  //
87  // q(x) + r(x,y) = 1*x^2 + (1*y^2)*(x^3)
88  // that is:
89  // x^2 + (y^2)(x^3)
90  //
91 
92  // Or evalue at specific point:
93  Poly rAtTwo = PolyAt(&r, 2);
94  // Now we assume the first variable is equal to 2
95  // So we calculate r(x,y) for x->2
96  // rAtTwo = r(2,y) = (2^3)*y^2 = 8*y^2
97 
98  // You can print them nicely:
99  PolyPrint(&rAtTwo);
100  // ^ Do it yourself and see what the output is
101 
102  // After everything we must free them all
103  PolyDestroy(&p);
104  PolyDestroy(&q);
105  PolyDestroy(&r);
106  PolyDestroy(&sum);
107  PolyDestroy(&rAtTwo);
108 
109  }
110 ```

112 # Polynomial calculator
113 
114 The calulator resides in `src/calc_poly.c`.
115 
116  It reads data from stdin line by line and:
117 
118 1. If the line starts with lower/upper case letter then it's assumed
119 
120  to be an input command.
121 
122 2. Otherwise the input is parsed as a polynomial
123 
124  The format of the polynomial is
125 
126  `(COEFF_1,EXP_1)+(COEFF_2,EXP_2)+...+(COEFF_N,EXP_N)`
127 
128  Where pair `(COEFF_N,EXP_N)` represents single monomial `(COEFF_N)*x^(EXP_N)`
129 
130  The coefficients can be numbers:
131 
132  `(1,3)+(2,4)` -> x^3 + 2x^4
133 
134  Or polynomials (in this case they depend on other variable):
135 
136  `((1,3)+(2,4),2)+(1,3)` -> (x^3 + 2x^4)*y^2 + y^3
137 
138  can also contain monomials with the same exponent:
139 
140  `(1,2)+(1,2)` -> 2x^2
141 
142  Or can be just numbers:
143 
144  `42` -> constant polynomial 42
145 
146 
147 All the polynomials are parsed and placed on top of the stack.
148 Then you can call one or more of the given operations
149 
150 to modify the stack contents:
151 
152 1. `ZERO` - pushes zero const polynomial onto stack
153 
154 2. `IS_COEFF` - check if the stack top is a constant polynomial?
155 
156 3. `IS_ZERO` - check if the stack top is a polynomial equal to 0?
157 
158 4. `CLONE` - Copy the polynomial on top of the stack and place it on the top.
159 
160 5. `ADD` - Adds two polynomials from the top of the stack.
161 
162  Then puts the result on the stack top.
163 
164 6. `MUL` - Multiplies two polynomials from the top of the stack.
165 
166  Then puts the result on the stack top.
167 
168 7. `NEG` - Changes the sign of the polynomial on the top of the stack.
169 
170 8. `SUB` - Substracts two polynomials from the top of the stack.
171 
172  Then puts the result on the stack top.
173 
174 9. `IS_EQ` - Checks if two top-most polynomials are equal.
175 
176 10. `DEG` - Checks the degree of the top-most polynomial.
177 
178  Degree of the polynomial is a highest exponent of variable
179 
180  encountered in the polynomial (assuming all variables are the
181 
182  same one).
183 
184  E.g. degree(x^2 + yx^2) = 3
185 
186 11. `DEG_BY [index]` - Checks the degree of the top-most polynomial
187 
188  with respect to the given variable. Variables are indexed from 0
189 
190  (0 means the main variable of polynomial).
191 
192  E.g.
193 
194  DegBy( ((1,2),3), 0 ) = DegBy( x^2*y^3, 0 ) = DegByY( x^2*y^3 ) = 3
195 
196  DegBy( ((1,2),3), 1 ) = DegBy( x^2*y^3, 1 ) = DegByX( x^2*y^3 ) = 2
197 
198 
199 12. `AT [value]` Evaluate the top-most polynomial in specified point
200 
201  and put it into stack.
202 
203  The polynomial is evalued for `MAIN_VARIABLE=[value]` where
204 
205  `MAIN_VARIABLE` is variable with index 0.
206 
207  E.g.
208 
209  At( ((1,2),3), 0 ) = At( x^2*y^3, 0 ) = AtY( x^2*y^3, 0 ) = 0
210 
211  At( (1,2), 2 ) = At( x^2, 2 ) = AtX( x^2, 2 ) = 4
212 
213 
214  The result of this command is always a polynomial of degree one smaller
215 
216  than the polynomial given.
217 
218 13. `PRINT` - Print the top-most polynomial.
219 
220 14. `POP` - Pop the top-most polynomial from the stack.
221 
222 15. `POW [exp]` - Calculate the top-most polynomial power and push into the
223 
224  stack. Obviously `[exp]` can be only a number!
225 
226 16. `COMPOSE [count]` - take top-most polynomail from the stack.
227 
228  (We will call it P)
229 
230  Then take `[count]` polynomials from the stack (let's call them Q1, Q2 ...).
231 
232  Then we know that `P = C_1*x_1^E_1 + C_2*x_2^E_2 + ...`
233 
234  so we substitute
235 
236  `x_1 -> Q1`
237 
238  `x_2 -> Q2`
239 
240  etc.
241 
242  if the `x_n` has got no matching `QN` then we assume `x_n -> 0`
243  Then we put result of such substitution onto the stack.
244 
245 17. `DUMP` - Print the stack contents.
246 
247 18. `CLEAN` - Clear the stack entinerely.
248 
249 19. `EXIT` - Force exit calculator.
